# Visual Arts Project

A project for **Reading Visual Arts** course.

## Author
- **Name:** Herald Carl Avila
- **Section:** BSCS 4B

## Description
This project is an interactive visual arts exploration that showcases creative digital artwork, including ASCII art representations. It demonstrates the intersection of technology and visual arts through web-based presentations.
